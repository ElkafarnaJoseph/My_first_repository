"LR-1" My first Repository

Відповіді на Кр : 

1) Git - система контролю версій, що дає змогу повернутися до конкретної версії файлів у майбутньому та вести роботу над файлами в команді.

2) Незатверджені зміни, проіндексовані і додані.

3) додає необхідний файл до репоиторію.Додати всі файли : $ git add

4) Перевіпяємо чи додані файли (Виделені зеленим кольором - додані , червоним - не додані );

5) Коміт- фксація тану сховища в певний момент часу , тобто фіксація змін ; 

6) Ініціалізація Git у папці - .git ; 

7) Назва коміту або суть зміни  

8) Зайти на свою сторінку в GitHub ==> Your profile ==> Repositories ; 

9) Вона використовує для копіювання файлів з історії або сцени в робочу директорію.
